# swissDAO smart contract

based on the ERC1155 standard

## v1

### Governance

* List of members
* Voting mechanisms for
 * role assignment
 * experience points increase

### Membership
* each member has
 * experience points
 * activity points
 * assigned roles

## v2

### Governance

* List of members
* Treasury using ERC20 wrapper
* Voting mechanisms for
 * grants and spendings
 * role assignment
 * experience points increase

### Membership
* each member has
 * experience points
 * activity points
 * assigned roles
